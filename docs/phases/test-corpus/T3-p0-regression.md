# T3 — P0 算子最小回归集

> 范围：`testdata/corpus/p0/`
> 依赖：T2
> 阶段目标：手写 + 抽取的 P0 算子最小回归集（~50 条），每条标注覆盖算子

## 顺序化子目标

### T3.S1 — P0 算子最小用例
- 产出：每 P0 算子 5-8 条手写用例（正常/边界/错误），覆盖 where/project/extend/take/sort/summarize/join。
- 验收：每算子用例数达标；边界（空表/NULL/大数）覆盖。
- 测试来源：手写。

### T3.S2 — 从 T2 抽取补充
- 产出：从 T2 抽取真实复杂度的 P0 查询补充。
- 验收：真实查询占比 ≥40%。
- 测试来源：T2 分类语料。

### T3.S3 — 元数据 tags 完整
- 产出：每条语料 tags 完整（覆盖的所有算子）。
- 验收：可按 tag 过滤跑测试。
- 测试来源：T1 schema。

### T3.S4 — 期望标注
- 产出：P0 子集手工标注 AST/IR 快照（expected_ast_snapshot / expected_ir_snapshot）。
- 验收：每条有期望快照或标注"待 golden 生成"。
- 测试来源：手写 + F4/I2 输出。

### T3.S5 — 索引与覆盖率
- 产出：`testdata/corpus/p0/index.yaml`（覆盖算子矩阵）。
- 验收：50 条覆盖全部 P0 算子；无遗漏。
- 测试来源：索引核对。

## 阶段产出物
- `testdata/corpus/p0/*.yaml`
- `p0/index.yaml`

## 风险与对策
| 风险 | 对策 |
|---|---|
| 覆盖算子不全 | T3.S5 矩阵核对 |
| 期望标注主观 | T3.S4 用 F4/I2 实际输出兜底 |
| 边界遗漏 | T3.S1 边界清单检查 |
